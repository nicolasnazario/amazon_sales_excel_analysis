# Amazon Sales Dashboard (Excel)

## Project Overview

This project analyzes Amazon sales data using Microsoft Excel to identify key revenue trends, top-performing product categories, and regional sales performance.

The dataset was cleaned and analyzed using Excel features such as pivot tables, calculated fields, and interactive charts.

## Key Business Questions

* Which product categories generate the most revenue?
* How do sales trends change over time?
* Which regions have the highest sales performance?
* How discounts and ratings affect revenue?

## Tools & Skills Used

* Pivot Tables
* Data Cleaning
* Data Aggregation
* Data Visualization

## Files in This Repository

amazon_sales_analysis.xlsx – Full Excel workbook containing the cleaned dataset, pivot tables, and dashboard.

## Key Insights

* Identified the highest revenue-generating product categories
* Analyzed monthly sales trends
* Compared regional sales performance
* Analyzed discount and rating effectiveness
